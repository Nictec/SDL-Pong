# SDL-Pong
A C++ project for learning SDL and developing a simple Game from scratch. 
## run the Game under a UNIX OS: 
``` bash 
## build under mac: 
cd source 
make 
make clean 
make run

## Build under Linux 
g++ main.cpp renderer.cpp window.cpp  -w <path_to_SDL2_executable> -o "SDL pong" 
## run the executable 
```
